# Introduction

Solid Auth is a TypeScript library that is build on top of <a href="https://start.solidjs.com/" target="_blank" rel="noreferrer">SolidStart</a> to easily add authentication to your SolidStart application. It is based on the SolidStart cookie session storage and includes many different useful providers.

## 🧪 Features

- OAuth2 & Auth0 Providers
- Custom Credentials Provider
- Ability To Login From Both Client & Server Side
- Using The SolidStart Cookie SessionStorage So You Can Manage The User Easily
